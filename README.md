<h2 id="-quick-start">
  <img src="https://readme-typing-svg.demolab.com?font=JetBrains+Mono:wght@700&size=22&duration=1&pause=1000&color=F59E0B&center=false&vCenter=true&width=300&lines=⚡+Quick+Start" alt="Quick Start" />
</h2>

Get MYiot running in **under 60 seconds** with Docker:

```bash
# 🐳 Docker (recommended)
docker run -d \
  --name myiot \
  --privileged \
  -p 8080:8080 \
  -p 8443:8443 \
  -v myiot_data:/data \
  myiot/myiot:latest

# 🚀 Or clone & run locally
git clone https://github.com/myiot/myiot.git && cd myiot
npm install          # Install frontend dependencies
pip install -r requirements.txt   # Install backend
npm run dev          # Start the dev server
```

```bash
# 🧪 One-liner installer
curl -fsSL https://get.myiot.io | bash
```

> 💡 **Pro Tip:** Add `--device /dev/ttyUSB0` for Zigbee/Z-Wave USB sticks.
